void DS3231M_Class::adjust ( )

sets the current date/time (Overloaded)

This is an overloaded function, if called with no parameters then the RTC is set to the date/time when the program was compiled and uploaded. Otherwise the values are set, but the oscillator is stopped during the process and needs to be restarted upon completion
